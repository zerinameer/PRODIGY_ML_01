# House Price Prediction
# About the Project

This project is a Machine Learning model that predicts house prices based on different features of a house. It was developed using Python and Scikit-learn as part of my learning journey in Machine Learning.
The main goal of this project is to understand how regression algorithms work and how they can be used to solve real-world prediction problems.

# Tools and Libraries Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

#Project Steps
- Loaded the housing dataset
- Explored the data to understand its structure
- Cleaned and preprocessed the dataset
- Selected useful features
- Split the data into training and testing sets
- Trained a Linear Regression model
- Predicted house prices
- Evaluated the model using different performance metrics


# Model Performance
The model was evaluated using:
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
These metrics helped measure how well the model predicts house prices.
